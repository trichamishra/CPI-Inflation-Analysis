## Overview

This project analyzes India's Consumer Price Index (CPI) data using Microsoft Excel to understand inflation trends, category-wise contributions, and the impact of major economic events. The dataset includes CPI values across multiple categories for both rural (CPI-R) and urban (CPI-U) populations.

## Objectives

* Identify contribution of broader CPI categories (Food, Energy, Transportation, etc.)
* Analyze year-on-year inflation trends
* Study food inflation and its monthly volatility
* Evaluate the impact of the COVID-19 pandemic on inflation
* Examine correlation between crude oil prices and CPI categories

## Dataset

* Source: Government of India (CPI dataset)
* Monthly CPI index values across categories
* Includes missing values handled using Excel techniques

## Tools & Excel Functions Used

* Microsoft Excel
* Pivot Tables
* Charts (Line, Bar, Pie)
* Data Cleaning & Preprocessing
* Moving Average (for missing values)
* Formulas:

  * Percentage Change
  * IF statements
  * AVERAGE
  * CORREL (for correlation analysis)

## Key Analysis Performed

### 1. Category Contribution

* Combined detailed categories into broader buckets (Food, Energy, etc.)
* Calculated percentage contribution using Excel formulas
* Visualized using Pie Chart

### 2. Year-on-Year Inflation

* Calculated YoY inflation using percentage change formula
* Created trend charts to identify peak inflation periods

### 3. Food Inflation Analysis

* Analyzed month-on-month changes using Excel formulas
* Identified highest and lowest inflation months
* Found key contributing sub-categories

### 4. COVID-19 Impact Analysis

* Compared inflation trends before and after March 2020
* Observed changes in essential categories like food and healthcare

### 5. Oil Price vs Inflation

* Used Excel CORREL function to analyze relationship between oil prices and CPI categories
* Identified categories most affected by oil price fluctuations

## Key Insights

* Food category has the highest contribution to CPI
* Inflation trends vary significantly across years
* Food inflation shows high volatility and seasonal patterns
* COVID-19 impacted inflation, especially in essential goods
* Transport and fuel categories show strong correlation with oil prices
